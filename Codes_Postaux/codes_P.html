Fond de carte des codes postaux

Les données géographiques sont fournies en projection Lambert 93 au format ESRI Shape et Mapinfo.

Liste des champs: id : identifiant du code postal à 5 chiffres lib : libellé du code postal dep : code du département surf : superficie (en km²) pop2010 : population estimée en 2010 men2010 : nombre de ménages estimé en 2010
